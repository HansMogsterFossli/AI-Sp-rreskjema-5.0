# Ørebro demo (Web Service) – v2.4.1

- **Light freedom**: Spørsmålene formuleres litt friere/mer naturlig, men scoring er den samme.
- AI gir **kort empati** og går rett til neste skjema-spørsmål (ingen ekstra oppfølgingsspørsmål).
- **Enter** sender svaret (du kan fortsatt klikke Send).

## Render
- Web Service
- Root Directory: (tomt)
- Build: `npm install && npm run build`
- Start: `npm run start`
- Env (valgfritt): `CHATAI_API_KEY`, `CHATAI_BASE_URL`, `CHATAI_MODEL`
